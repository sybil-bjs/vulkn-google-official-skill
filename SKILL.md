# Google Official Skill

This skill provides direct access to Google Workspace (Gmail, Calendar, Drive, Sheets, Docs) using the official `googleapis` library and the VULKN Service Account.

## Setup
1. Ensure the Service Account key is at `~/.config/gog/vulkn-service-account.json`.
2. Run `npm install googleapis`.

## Capabilities
- **Gmail:** List, read, and send emails.
- **Calendar:** Manage events and schedules.
- **Drive:** Search, upload, and manage files.
- **Docs/Sheets:** High-fidelity document and spreadsheet operations.
